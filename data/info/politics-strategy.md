# 政治 75+ 备考策略

## 核心逻辑

政治 100 分 = 选择题 50 分 + 分析题 50 分。

**选择题拉开差距，分析题靠肖四突击。** 选择题没法突击，必须扎实过知识点；分析题最后背肖四基本够用。

## 最优方案：徐涛的课 + 肖秀荣的书 + 腿姐的技巧

### 基础阶段（7月 → 8月）每天 1-1.5h

- 看徐涛强化班（马原别倍速，听不懂倒回去再听一遍）
- 搭配肖秀荣《精讲精练》，看一章刷一章对应 1000 题
- 只做选择题，错题标记原因

### 强化阶段（9月 → 10月）每天 1.5-2h

- 二刷《1000 题》错题 + 标记的重点题
- 腿姐技巧班（学多选题排除法、关键词法）
- 用腿姐/徐涛《冲刺背诵手册》开始记忆核心考点

### 冲刺阶段（11月）每天 2h

- 肖八到手后只做选择题，两天一套
- 做完总结错题+背对应知识点
- 肖秀荣《形势与政策》补时政

### 背诵阶段（12月）每天 2.5-3h

- 肖四到手后 **大题全文背诵**（至少背 3 遍）
- 同时认真做肖四选择题
- 腿姐/徐涛押题卷选择题刷一遍查漏补缺

## 学习顺序

马原 → 史纲 → 毛中特 → 新思想 → 思修法基 → 时政

- 马原最难，需要理解，放最前面
- 先学史纲为毛中特打基础
- 时政不用提前学，11 月看肖秀荣《形势与政策》即可

## 推荐资料（按购买顺序）

| 资料 | 时间 | 用途 |
|------|------|------|
| 肖秀荣《精讲精练》 | 7月 | 教材，覆盖所有大纲考点 |
| 肖秀荣《1000题》 | 7月 | 刷选择题，至少 2 遍 |
| 徐涛强化班（网课） | 7-8月 | 理解知识点 |
| 腿姐技巧班（网课） | 9月 | 选择题解题技巧 |
| 腿姐/徐涛背诵手册 | 10月 | 浓缩考点快速记忆 |
| 肖八（八套卷） | 11月 | 冲刺模拟，只做选择 |
| 肖四（四套卷） | 12月 | **大题必背** |

**省钱方案**：最少保证《精讲精练》+《1000 题》+ 肖四肖八。网课可找免费资源。

## 关键提醒

- 单选正确率目标 70%+，多选 50-55%
- 肖四大题不要死背原文，记关键词+逻辑框架
- 选择题靠理解（7-10月的核心任务），分析题靠背诵（12月）
- 近两年出题风格变灵活，死记硬背效果下降
- 肖老原话：分析题可以突击，选择题不行
